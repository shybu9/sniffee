### PRE-REQUIREMENTS 
* python3 :
```bash
    sudo apt-get install python3
```



* script_requirements :
```bash
    sudo pip3 install -r requirements.txt
``` 
### DOWNLOADING_SCRIPT :
```bash
    sudo git clone https://github.com/shybu9/sniffee
```
## USAGE 

### SYNTAX : 
```bash
   sudo python3 sniffee.py --iface <network_interface>
```

### EXAMPLES :
`$ sudo python3 sniffee.py --iface eth0 `
